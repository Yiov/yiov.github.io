# 微软Azure·云希音色配音

更新时间

最近更新：2022-7-31

短视频配音必备！我们抖音常见的电影解说配音都出自它，实际是一个文本转语音功能，而 云希 就是最火的一个音色

我们来看看怎么使用

直达：https://azure.microsoft.com/zh-cn/services/cognitive-services/text-to-speech/#featuresopen in new window

# Azure

微软Azure官网：https://www.azure.cn/open in new window

产品-AI-语音服务

进入页面后下拉，中间位置，找到为应用提供语音下的详细了解文本转语音

进入新页面后，下拉到底部就能看见了

我们音色选择云希，左边对话框打上你的台词，播放即可

# 使用工具

那么问题来了，只能听啊，不能下载

任选一种方法，使用即可

QQ的屏幕录制（笨方法）

适合懒人和台词量小的，默认的快捷键是ctrl+alt+s

随便拉个框了开始，再点播放录制完整语音，但是保存的是mp4需要我们单独提取一下音频即可使用

油猴插件（推荐）

说明

未安装的看 油猴安装及使用教程

完成安装后，我们还需要在Greasy Forkopen in new window搜索安装 Azure 插件，据说这3个都好用，我们就选第一个安装吧

安装完成没有变化，刷新下网页看看，显示重新安装就是成功了

我们进入Azure刷新页面看看，就多了一个 下载按钮 ，下载即可

说明

如果你有导入字幕文件转换的需求，建议用第三种方法
