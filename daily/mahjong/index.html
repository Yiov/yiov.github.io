# 台州麻将玩法小结

# 基本知识

# 牌数

万牌：1-9万 🀇 🀈 🀉 🀊 🀋 🀌 🀍 🀎 🀏

条牌：1-9条（索） 🀐 🀑 🀒 🀓 🀔 🀕 🀖 🀗 🀘

筒牌：1-9筒 🀙 🀚 🀛 🀜 🀝 🀞 🀟 🀠 🀡

风牌：东、南、西、北 🀀 🀁 🀂 🀃

字牌：红中、白板、发财 🀄 🀅 🀆

34种牌，各4张，共136张牌

# 堆牌

从打散的136张牌中，每家将麻将堆成17堆，每堆2层，34张牌

说明

现在都是麻将机了，自动洗牌堆牌

# 筹码

无论筹码是何种形式，纸牌也好硬币也罢，都是一个计算工具

100x4个 + 50x1个 + 10x4个 + 1*10个

筹码共计：500

输完即结束所有牌局，开始计算输赢金额，开局先定好每番金额，0.5元/胡 或者更高

建言

小赌怡情，大赌伤身，享受生活，请勿沉迷赌博

# 骰子和庄家

骰子有两粒，每粒6点，最大点数和为12

以首次掷骰子的人为基准1，右手逆时针转圈，其他三人分别为2 / 3 / 4，总和依此类推如下

庄(东)：1、5、9 北（上家）：2、6、10 西（对家）：3、7、11 南（下家）：4、8、12

骰子的用处主要是确定摸牌位置

首次庄家（东风家）也可由掷骰子决定，比点数大小，之后庄家采取轮庄方式

坐庄：庄家掷骰子确定从哪一方拿牌，确定拿牌的一方再次掷骰子确定摸牌的位置

比如：
庄家掷骰子的和为7，则从对家(西风)家摸牌，
西风家再掷骰子的和为5

那么两次的总和为12，从西风家右手边往左数到12堆牌，则庄家从第13堆开始摸牌

连庄：每次牌流结束，如果庄家胡牌，就是连庄

下庄：非庄家胡牌，由原庄家的下家(右手)继任

流局：场上所有牌摸完，无人胡牌，庄家下庄，下家继任

# 拿牌和得牌

拿牌：由庄家开始拿牌，每次2堆(4张)，然后由下家轮次拿牌，每人3轮，各家12张

跳牌：庄家再摸第1张和第5张，其他人不跳牌仅摸1张

每局牌数：首轮庄家14张牌，其他人13张，手中牌永远是单数，否则无法胡牌

得牌：每局各家摸牌结束后的第一张牌即为得牌，也叫万能牌，它可以当作任何牌(包括本身)，但不可单独使用

当前翻过来的得牌不可用于牌局，另外的3张可用于牌局

庄家先手，不摸牌直接出牌，其他人摸一张再打一张，直到胡牌

# 牌型

顺子：3张连续数字且种类相同的牌 如：123万 🀇🀈🀉

对子：是由2张相同牌 如：2个8万 🀎🀎

刻子：是由3张相同牌 如：3个东风 🀀🀀🀀

暗刻：自己摸3张相同牌，未明牌

杠牌：是由4张相同牌 如：4个东风 🀀🀀🀀🀀

暗杠：自己摸4张相同牌，未明牌

白板：可以作为自身，也可以做为当前的得牌，但不能成为得牌

如：得牌为🀄，那白板只能当作 🀆 和 🀄

得牌不小心打出去就当做废张，不能吃碰，得牌不能参与吃/碰/杠

# 玩法

吃牌：只能吃上家的牌，若上家打出的牌跟自己手上的2张牌可组成顺子，可选择是否吃牌

吃牌的人不摸牌

如：上家打 🀇 ，自己有 🀈 🀉

碰牌：可以碰任何一家的牌，当别人打出一张牌时，自己手上有2张相同牌，可选择是否碰牌。

碰牌的人不摸牌，且在本轮里，不能弃先碰后(第一个先打的没碰，第二个接着打也不能碰），下一轮不受影响

如：其他人打 🀇 ，自己有两个 🀇🀇

杠牌：可以杠任何一家的牌，分为暗杠和明杠，杠牌的人从最尾端摸一张补牌

暗杠：自己摸牌4张，未明牌就叫暗杠(亮1张牌)

明杠：自己摸牌3张，杠别人1张(或者先碰1张再摸到1张)，明杠的情况下如有其他人刚好胡这张杠字，是可以胡的，就叫抢杠胡

听牌：自己的牌就差一张胡牌，即算听牌

胡牌：胡牌的基础必须有一对角牌(任意一对)，剩下的牌中没有任何一张单牌，即可胡牌

可以自摸也可以捉铳，杠开算自摸，抢杠算捉铳，同样也不能弃先胡后

危险牌：当红中发财白板未打出时，它们统称即为危险牌，若自己未听牌，打出了中发白导致别人胡了，则要包场替所有人付钱

# 胡数及台数

对子：0胡

门风对：2胡 各家相对应的风牌，如：庄家有一对东风

中发白对：2胡

明刻：2胡，幺九刻(包括风牌和字牌)，4胡

暗刻：4胡，幺九刻(包括风牌和字牌)，8胡

明杠：8胡，幺九杠(包括风牌和字牌)，16胡

暗杠：16胡，幺九杠(包括风牌和字牌)，32胡

# 额外奖励胡数：

自摸：2胡 对对胡：4胡 嵌档胡：2胡 胡牌奖励：10胡(底胡)

清一色：3台 手中牌全部为同一花色(万/条/筒)，含吃了的牌 如果有白板，那么财神牌也是同一花色(万/条/筒)才能算清一色，不然只能算混一色，

混一色：1台 手中牌有一对/刻子/杠的任意风牌或者字牌，剩余牌全部为同一花色(万/条/筒)

门风刻和字牌刻(含杠)：1台 可以由财神牌代替，但至少得有1张

无得：1台 没有任何一张财神牌或者财神牌能还原 还原的情况中不能含白板，且未胡牌的人不计算无得的奖励

# 胜负规则

不管是自摸，还是捉铳，未胡牌的3人，都要按胡牌人的总胡数来计算

胡者总胡数：（基础胡数＋10）＊2的台数次方

基础胡数＝所有牌型的胡数之和(不包括奖励的10胡)

庄家：按胡者的总胡数计算

闲家之间：按胡者总胡数的一半计算

勒子：全员按封顶100胡计算

比如： 闲家A胡牌，20胡 庄家：10胡 闲家B：6胡 闲家C：20胡

战况 庄家：输20胡，赢闲家B 4胡，输闲家C 10胡，共计输28胡

闲家A：赢庄家20胡，赢闲家B（20-6）/2=7胡，赢闲家C（20-20）/2=0胡，共计27胡

闲家B：输闲家A（20-6）/2=7胡，输庄家10-6=4胡，输闲家C（20-6）/2=7胡，共计输18胡

闲家C：赢闲家A（20-20）/2=0胡，赢庄家20-10=10胡，赢闲家B（20-6）/2=7胡，共计赢17胡

验算：两边的输赢相等 闲家A（+27胡）=庄家（-28胡）+闲家B（-18胡）+闲家C（+17胡）

庄家胡牌，20胡，其他3位闲家输20胡

特例：流局不算胡数及输赢

承包： 承包者的胡数清0，并且承包台面上所有人（包括非胡家之间）的胡数

一、当1个玩家台面上有了3比或者4比同色的牌，不管谁用这一色的牌点炮，点炮者全包。台面上3比一色的牌加上另1比其他的牌不用承包。 二、当能胡的时候不胡，打中发白，承包。（除了清一色） 三、当玩家打生张中发白时（白板是硬张）点炮，根据下面情况： a、如果此玩家没有差1张牌听张，承包。 b、如果此玩家差1张牌听牌，如果打某张牌（中发白除外）后能上听的叫数大于等于目前上听的叫数，承包。如果手里多张单张的中发白，必须先打熟张。（比如财神是1万，用白板2万吃3万，白板还是生张。） C、如果打的这张中发白，如果不打这张能听牌，承包。 d、如果听牌，如果不是清一色，则比较另外听牌的张数，如果另外听牌的张数小于等于打中发白时听牌的张数，承包。 四、如果打的这张白板是软张，根据下面情况： a、如果此玩家没有差1张听牌，承包。b、如果此玩家差1张牌听牌，如果其他牌有孤张（财神不算），承包。否则比较打其他牌能上听的张数，只有在打白板时能上听的叫数最多时不承包。 C、如果把听牌打成不听牌，承包。d、打白板只有在上听叫数最多并且胡牌张数是多时不用承包

3、流局 指抓完指定的牌，再打出一张，可出任意牌，但是还没有人胡牌，称为流局；流局不计任何分 。

另流局留牌的张数算法是如没有人杠，则留7堆牌，即14张，如有人杠过，则杠一次流局张牌多加两墩（4张牌），并且杠的时候摸走了那墩不算。比如有人杠了一次，则流局的张数为19张了，杠了二次后，则流局的张数为11墩，22张。 4、补牌 杠牌的时候，则从尾部从上往下轮流补牌。 5、基本打法 可吃，可碰，可杠，可放炮胡、自摸胡。字不可成顺。 6、得 即起始发牌结束后，下一张牌翻开为“得”。“得”可做万能牌用，翻开的“得”摆放在牌墩上。白板可替代“得”牌原来牌值，但没有万能牌作用。除非“得”为白板。比如1万为“得”，我手上一个白板，一个二万，上家打三万，我可以拿白板和二万去吃三万。所以白板可替“得”那张牌也可能还原成白板 ---白板还原为“得”时，只能组合成顺子的情况下，非顺子情况下，算白板原有牌值。 7、腊子 当所胡牌型得分超过100胡时，最多赔付100胡。 8、放弃过的胡张同轮内不能再胡，一轮过后照常 9. 全包 1)当一家做清一色的牌型并吃上家三次或三次以上时，当家胡牌，则上家形成全包。如果当家胡中发白任一张，其他三家放铳中发白的生牌，当家胡，放铳一家听牌时全包，需负担另外两家应付的胡牌分。 2)中发白三者任意两者碰了，当其中一家打剩下的中发白一者碰胡，则放銃家形成全包未胡牌玩 台州麻将胡牌的基本牌型 1)11、123、123、123、123 2)11、123、123、123、111(1111，下同) 3)11、123、123、111、111 4)11、123、111、111、111 5)11、111、111、111、111 台州麻将只有自摸胡、放铳胡、清一色、混一色、坎张胡、碰碰胡。 胡牌者和未胡者间，按胡牌者的胡数赔付，未胡者和未胡者之间按两者胡数差赔付。胡家可额外获得的奖励均为10胡。 胡牌者和未胡者间算胡时，当胡数没有腊子（封底）的情况下，非胡的闲家只出非胡庄家一半的胡数 自摸胡：+2胡 放铳胡：即放炮胡的意思 清一色：3台 混一色：1台 坎张胡：即胡卡中间的那张牌，+2胡。 对对胡：即碰碰胡 +4胡
