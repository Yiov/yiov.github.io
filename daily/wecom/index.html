# 企业微信通知推送

更新时间

最近更新：2022-9-3

通过自建应用，利用推送工具，可无需下载企业微信，微信即可接收通知

注意

使用企业微信做推送，也有一定几率被封号，如果被封将无法再使用企业微信，建议使用小号！

# 1.注册

企业微信官网：https://work.weixin.qq.com/open in new window

点击 ⽴即注册

说明

注册有2种：企业版 和 教育版，任选其一即可！

企业版 需要加入企业并关注插件

教育版 只需要邀请进班级即可

# 1.1 企业注册

企业名称随意，管理员姓名真实，扫码绑定

这样就注册成功了

# 1.2 学校注册(推荐)

企业名称随便，行业类型必须教育-学前教育，员工规模101-200人

说明

超200人就要认证费了，200以下免费

其他信息真实的，绑定微信注册成功

# 2. 获取企业ID

我的企业 - 企业ID

# 3.常规设置

进入管理后台，关闭这里的自动提醒成员使用，而且这里提示有1人未加入，我们下载企业微信app并登陆

在 聊天管理 成员创建群聊上限改成最低，关闭 消息阅读状态、部门群 、全员群

在 通讯录管理 中关闭通讯录访问权限，对所有⼈隐藏，主要是保护个人隐私

在 工作台管理 中关闭成员可见的 第三方入口 和 上门服务入口

在 微信插件 中保存邀请关注⼆维码，关闭微信插件底部快捷打开

在 申请加⼊设置 中关闭通过企业查找加入

说明

如果不想别人乱加，可以开启 加入企业管理员审核

在 设置 中关闭使⽤提醒

# 4.获取应用ID和Secret

应⽤管理 中创建应⽤

上传LOGO、填写应⽤名称，选择部门，创建

记下 AgentId ，点击 查看 获取 Secret

说明

手机登录企业微信app查看

# 5.关于推送

# 5.1 青龙推送（企业版）

查看 通讯录 中的帐号 ，微信插件是否 已关注 状态

说明

未关注需要扫描 我的企业 - 微信插件 -邀请关注 中的⼆维码

到 发送消息 中 发消息,素材库-图片-上传图⽚

上传完成后，点击图标，右键-复制图片地址

media_id 就是图⽚ID，这个类似微信公众号的封⾯图

我们来看看青龙的配置要求

## 企业微信应用
## 参考文档：http://note.youdao.com/s/HMiudGkb
## 下方填写素材库图片id（corpid,corpsecret,touser,agentid），素材库图片填0为图文消息, 填1为纯文本消息
export QYWX_AM=""

我们按照对应的变量填写即可

export QYWX_AM="企业ID,应⽤Secret,账号(@all),应⽤ID,图⽚ID(选填)"

# 5.2 家校推送（教育版）

应用下填写 应用主页，必填！网址就随意

注意

一定要填写，否则会无法使用

家校通知 - 配置 -家长可使用的应用

添加应用

选择 刚出创建的应用

说明

如果这里没有创建的应用，是因为应用没有设置应用主页

小字也提示了：仅支持 设置了工作台主页 的应用和学校已关联的小程序

家校沟通-家校通讯录

说明

没有家校沟通，说明你注册，就没有选教育

删掉学段，我们自己自定义添加

说明

同样我们也可以自定义班级

点 邀请加入 ，保存二维码发给别人，就可以摇人了

说明

或者草料识别二维码后，提取连接

姓名随便，手机要微信绑定的，提交

微信就多了一个 学校通知
