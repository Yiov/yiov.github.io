# 使用

有了工具，我们还需要节点(流量)

说明

第一次使用，建议随便挑一个有新户注册送流量的机场，好用再花钱买

请参照 机场channel大全一览表

挑好机场并注册，然后我以安卓的小猫咪 Clash 演示

说明

其他应用请参考机场的使用说明文档
