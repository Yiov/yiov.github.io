# 快速上手

更新时间

最近更新：2023-7-15

搭建版本：v2.0-beta.66

那我们就来学习如何的搭建vuepress，就是你现在看到的这种文档网站

# 前期工作

# 工具

说明

已经安装 或者 熟练了，可以不用看此步骤

  • 必装：安装nodejs在新窗口打开，不想要文字删掉留空即可

  • 建议安装：安装vscode在新窗口打开，不想要文字删掉留空即可

  • 可选安装：安装git在新窗口打开，不想要文字删掉留空即可

# pnpm/yarn

通过 npm 安装 yarn在新窗口打开，不想要文字删掉留空即可 或者 新出的pnpm在新窗口打开，不想要文字删掉留空即可

#安装pnpm
npm install -g pnpm
#查看版本号
pnpm -v

#安装yarn
npm install -g yarn
#查看版本号
yarn -v

# 创建目录

win键+R键，调出命令运行框，输入 cmd

先进入任意盘符，比如 F 盘

#盘符可以自定义 回车进入
f:

再创建文件夹名

#目录可以自定义
mkdir vuepress

#目录和上面保持一致
cd vuepress

说明

这样我的目录路径为 F:\vuepress

觉得麻烦的可以直接电脑创建

# 开始安装

我们在目录上方的地址栏输入 cmd 可以快捷打开

# 初始化

#初始化pnpm
pnpm init

#初始化yarn 一路回车
yarn init

#或者用一键y命令
#yarn init -y

#初始化npm
npm init

# 安装依赖

vite说明

从v2版本开始，默认使用的是 Vite在新窗口打开，不想要文字删掉留空即可 打包工具，webpack在新窗口打开，不想要文字删掉留空即可编译慢，不建议使用

#安装 新增webpack/vue和@vuepress/client
pnpm add -D vuepress@next @vuepress/client@next vue webpack

#安装含Vite
yarn add -D vuepress@next

#卸载
#yarn remove vuepress

#安装含Vite
npm install -D vuepress@next

# 脚本命令

在 package.json 中添加一些 scripts 命令，保存

命令解析

执行命令 docs:dev，就会运行 vuepress dev docs

执行命令 docs:build，就会运行 vuepress build docs

  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }

 
 

# git忽略项

添加.gitignore文件，主要用于上传到gitee/github

注意

.gitignore 里放的是上传时git的忽略项

建议先使用cmd，使用vscode有可能会出现乱码

#将依赖文件添加到.gitignore文件中
echo node_modules >> .gitignore

#将临时目录添加到.gitignore文件中
echo .temp >> .gitignore

#将缓存目录添加到.gitignore文件中
echo .cache >> .gitignore

#将静态目录添加到.gitignore文件中
echo dist >> .gitignore

Github上传未忽略dist文件夹

原因：vscode输入命令导致编码错误

解决：Github Desktop - Repository settings - ignored files

输入我们忽略的dist文件即可，save保存即可

node_modules
.temp
.cache
dist

# 创建文档

注意

建议先使用cmd，使用vscode有可能会出现乱码

实在搞不定，就手动新建一样的

创建你的第一篇文档

mkdir docs
echo '# Hello VuePress' > docs/README.md

# 本地启动

在本地启动服务器，来开发你的文档网站

如何退出

本次执行的其实就是我们在 脚本命令 中的命令

另：ctrl+c 即可退出开发模式

#开发环境
pnpm docs:dev

#开发环境
yarn docs:dev

#开发环境
npm docs:dev

VuePress会在本地启动一个8080端口的热重载开发服务器

说明

当你修改文件时，浏览器中的内容也会自动更新

http://localhost:8080

成功看到页面，就说明我们的vuepress就搭建完成了

注意

接下来我们可以关闭cmd，全程用vscode了
