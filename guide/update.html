# 更新及卸载

更新时间：2023-11-17

# 更新

# 安装ncu

注意

已经安装过的可以直接看 更新包 步骤

由于直接用命令更新，不能改变 package.json 的内容

我更推荐用 npm-check-updates，即 ncu

npm i -g npm-check-updates

ncu可以可以检查出需更新的软件包

ncu

# 更新包

更新模块包版本信息

#即运行ncu -u to upgrade package.json
ncu -u

说明

这时候我们的 package.json 里的版本都按最新的填写完毕了

我们直接用命令更新安装依赖即可

pnpm install

yarn install

bun install

# 卸载

卸载就比较简单了，在 package.json 中找到包名即可

pnpm uninstall 包名

yarn uninstall 包名

bun uninstall 包名

比如

pnpm uninstall vue

其中vue 就是包名
