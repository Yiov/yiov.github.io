# 利用MacDirtyCow无限自签

更新时间

最近更新：2023-3-16

提示：巨魔与越狱用户可以无视！

# 前身

说明

MacDirtyCow漏洞，也就是 CVE-2022-46689 漏洞

  • Google Project Zeroopen in new window 的 Ian Beer 提交了这个漏洞，它可以使App能够以内核权限执行任意代码

  • 大神 @Zhouwei Zhangopen in new window 利用此漏洞发布了项目 MacDirtyCow Demoopen in new window，通过 MacDirtyCow 可以制作一些特殊的应用程序，允许重写沙盒，覆盖文件，且无需越狱

  • 2023-1-31 大神 @XsF1refopen in new window 发推通过使用此漏洞修补 /user/libexec/istalld 文件，可以移除Apple ID安装3个限制

  • 2023-2-4 @Zhouwei Zhangopen in new window 发布 WDBRemoveThreeAppLimit

# 原理

基本上都是通过交换同等大小文件的方式，覆盖文件实现

说明

覆盖的文件体积越大，越占用内存

所以，利用此漏洞本身就一定会出现内存占用情况，手机变卡、小组件无法使用等情况，只能通过重启后释放内存

# 适用范围

  • 支持 iOS 14.0 - 15.7.1 及 iOS 16.0 - 16.1.2 系统

  • 不支持 iOS 15.7.2 和 iOS 16.2 及更高的系统

  • IOS 14系统：不推荐使用

  • 巨魔/越狱用户：不推荐使用

# 准备工作

  • 安装：牛蛙助手

  • 下载：WDBRemoveThreeAppLimit：突破3个自签限制open in new window

  • 下载：Blacklist：一键修复证书失效open in new window

# 破除自签限制

下载WDBRemoveThreeAppLimit：https://github.com/zhuowei/WDBRemoveThreeAppLimit/releasesopen in new window

打开牛蛙助手，右上角+ -> 导入资源

选择刚下载好的 WDBRemoveThreeAppLimit，并导入应用库

说明

Safari浏览器下载的默认在文件-icloud云盘-下载里

选择 WDBRemoveThreeAppLimit 签名

立即签名- 安装

打开 WDBRemoveThreeAppLimit ，进入会有个提示，点好

点击 go，提示成功后重启一下手机

我们继续用牛蛙自签，可以看到我已经装了5个了

# 修复证书失效

同样的方法我们下载安装 Blacklist：https://dzp.lanzouy.com/im6L30sv8mojopen in new window

说明

主要用于修复右图这种，证书过期打不开的情况

进入的提示点好，在点击修复 Fix Blacklist 即可
