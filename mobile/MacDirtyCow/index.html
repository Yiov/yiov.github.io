# 利用MacDirtyCow无限自签

更新时间

最近更新：2023-4-16

# 前身

说明

MacDirtyCow漏洞，也就是 CVE-2022-46689 漏洞

由 Google Project Zeroopen in new window 的 Ian Beer 提交了这个漏洞，它可以使App能够以内核权限执行任意代码

大神 @Zhouwei Zhangopen in new window 利用此漏洞发布了项目 MacDirtyCow Demoopen in new window

通过 MacDirtyCow 可以制作一些特殊的应用程序，允许重写沙盒，覆盖文件，且无需越狱

# 原理

基本上都是通过交换同等大小文件的方式，覆盖文件实现

说明

覆盖的文件体积越大，越占用内存

所以，利用此漏洞本身就一定会出现内存占用情况，手机变卡、小组件无法使用等情况，只能通过重启后释放内存

建议

若无必要，使用 巨魔 或者 越狱 是更好的选择

# 适用范围

  • 支持 iOS 14.0 - 15.7.1 及 iOS 16.0 - 16.1.2 系统

  • 不支持 iOS 15.7.2 和 iOS 16.2 及更多的系统

  • IOS 14系统：不推荐使用 (部分App不支持且无法完美使用)

  • 越狱用户：不推荐使用

# 无限自签

  • 安装：牛蛙助手

  • 下载：WDBRemoveThreeAppLimitopen in new window
