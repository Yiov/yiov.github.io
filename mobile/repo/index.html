# 巨魔IPA应用

注意

一些软件使用了蓝奏云上传，导致下载的不是IPA包，而是直接被安装

为避免这种情况，请优先安装 IpaDownloadTool：IPA下载器open in new window

然后在 IpaDownloadTool 内，下载蓝奏上传的IPA
