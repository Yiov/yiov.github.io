# Telegram的注册及使用

更新时间

最近更新：2021-10-18

俗称TG，作为聊天软件中的翘楚，自然是微信无法企及的高度

说明

全程需要翻墙使用！

参考之前的文章☛科学上网工具及使用方法

# 1.注册

TG官网：https://telegram.org/appsopen in new window

根据自己的客户端下载软件

打开App 点下一步啊

说明

开梯子啊，不然不跳转！

输入自己的手机号，注册账号

输入验证码

名字随便，反正后面可以改

这样就注册成功了

加群链接一般是 https://t.me/***

说明

TG里使用直接 @*** 即可

# 汉化

系统自带的语言没有，用翻译包即可

APP搜索 zh_CN 或者 跳群：https://t.me/zh_CNopen in new window

# 隐私

非常重要，手机号暴露在外，有些内心阴暗的就要轰炸你

设置-隐私和安全-手机号码-不允许任何人

# 常见问题

# 1.进群啥也没有，看不到群

打开登入网页版：https://web.telegram.orgopen in new window

设置Settings-Privacy and Security - 下拉底部 勾选 Disable filtering

说明

关掉网页，重新打开APP即可恢复正常

# 2.不能和其他人进行私聊

搜索机器人 Spam Info Bot，按步骤点击后，最后输入 accident

# 3.不使用账号是否会注销

会，默认6个月不发消息自动注销，可以在设置-隐私和安全里延长为1年

# 4.如何注销账号

警告

注销后账号群组消息，全部消失！请确保无其他损失！

加群后有些群有防机器人验证，超时后会永久踢出，可以注销ID后重新加入

建议还是直接找管理解除省事

进入注销网站：https://my.Telegram.org/deleteopen in new window

在 Your Phone Number 栏输入要注销账号的手机号

打开电报APP，官方有个英文code验证码消息，复制

在 Confirmation Code 栏粘贴收到的验证码，再点 Sign In登录

在 Why are you leaving 栏随便写

点 Delete My Account 删除账户，YES

注销成功会提示，APP的账号也自动退出登录，重新注册即可
